📖 Overview
This is a high-performance frontend application built for the modern web. It transitions away from static layouts into cinematic digital experiences. Using GSAP and ScrollTrigger, I’ve engineered a UI that responds to user interaction with fluid, timeline-based animations.

"We make it happen." – Bridging the gap between a founder's vision and a pixel-perfect reality.

🛠️ Technical Implementation
⚡ Creative Development (GSAP)
Scroll-Driven Orchestration: Utilized ScrollTrigger to pin sections and scrub through complex multi-stage timelines.

Staggered Reveals: Custom menu and "Showreel" entrance using gsap.fromTo patterns for a luxury feel.

Performance Optimization: Leveraged gsap.context() for clean memory management and proper animation cleanup in the React lifecycle.

🏗️ Architecture
Next.js & React: Implementing Server Components for faster initial loads and SEO-friendly rendering.

Tailwind CSS: A design-system-first approach to ensure responsiveness across all viewports (Mobile-First).

TypeScript: Type-safe props and state management to ensure long-term codebase maintainability.
